Sistemi di crittografia con l'utilizzo di Python

Di seguito la spiegazione del primo programma in cui si esegue una crittografia a chiave simmetrica: from cryptography.fernet import Fernet: Importa la classe Fernet dal modulo cryptography.fernet. Fernet è una classe che fornisce implementazioni per crittografare e decrittografare dati utilizzando l'algoritmo di crittografia Fernet. key = Fernet.generate_key(): Genera una chiave casuale utilizzata per crittografare e decrittografare i dati. La chiave generata sarà un oggetto di tipo bytes. cipher = Fernet(key): Crea un oggetto Fernet utilizzando la chiave generata. Questo oggetto verrà utilizzato per crittografare e decrittografare i dati. data = b"Il mio messaggio segreto": Definisce i dati da crittografare. In questo caso, viene utilizzata una stringa in formato di byte. encrypted_data = cipher.encrypt(data): Crittografa i dati utilizzando l'oggetto Fernet creato in precedenza. Il risultato crittografato sarà un oggetto di tipo bytes. print("Dati crittografati:", encrypted_data): Stampa i dati crittografati. decrypted_data = cipher.decrypt(encrypted_data): Decrittografa i dati crittografati utilizzando l'oggetto Fernet. Il risultato decrittografato sarà un oggetto di tipo bytes. print("Dati decrittografati:", decrypted_data): Stampa i dati decrittografati. Questo codice illustra un esempio di crittografia e decrittografia utilizzando l'algoritmo Fernet. La crittografia simmetrica consente di crittografare e decrittografare i dati utilizzando la stessa chiave. 


from cryptography.fernet import Fernet

# Genera una chiave casuale
key = Fernet.generate_key()

# Crea un oggetto Fernet con la chiave generata
cipher = Fernet(key)

# Dati da crittografare
data = b"Il mio messaggio segreto"

# Crittografa i dati
encrypted_data = cipher.encrypt(data)

print("Dati crittografati:", encrypted_data)

# Decrittografa i dati
decrypted_data = cipher.decrypt(encrypted_data)

print("Dati decrittografati:", decrypted_data)
